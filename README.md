# Customer-Behaviour-Analysis-Dashboard
This project analyzes customer shopping behavior using SQL, Python, and Power BI to uncover insights about purchasing patterns, revenue distribution, subscription trends, and demographic behavior.

The repository includes:
                          *Interactive Power BI Dashboard
                          *SQL queries for data extraction and analysis
                          *Python Jupyter Notebook for data cleaning & EDA
                          *Raw customer shopping dataset

The goal of this project is to transform raw transactional data into actionable business insights.
